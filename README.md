# LYRICS-EDITZ
### Description
LYRICS-EDITZ is a web application that allows users to edit and manage lyrics of their favorite songs. It's built using JavaScript, HTML, and CSS, with the help of popular libraries like React and React Router.

### Installation

To install LYRICS-EDITZ, follow these steps:

```bash
# Clone the repository
git clone https://github.com/Aniketjamunde9146/LYRICS-EDITZ.git

# Navigate to the project directory
cd LYRICS-EDITZ

# Install dependencies
npm install

# Install dev dependencies
npm install --dev
```

### Usage

To start the development server, run the following command:

```bash
npm run dev
```

This will start the development server and make the application available at `http://localhost:3000`. You can then access the application in your web browser and start editing lyrics.

### Features

* Edit and manage lyrics of your favorite songs
* Search and filter lyrics by song title or artist name
* Save and load edited lyrics for future reference
* Preview edited lyrics in a readable format

### Tech Stack

* Primary language: JavaScript
* Languages used: JavaScript, CSS, HTML
* Dependencies: fonts, google, react, react-dom, react-router-dom
* Dev Dependencies: @eslint/js, @types/react, @types/react-dom, @vitejs/plugin-react, eslint, eslint-plugin-react-hooks, eslint-plugin-react-refresh, globals
